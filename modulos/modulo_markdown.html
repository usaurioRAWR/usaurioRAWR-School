Markdown

(abreviado comúnmente md)

Markdown es un lenguaje de marcado que permitir crear contenido de una manera sencilla y rapida, usando texto plano.

Es soportado por muchas plataformas como; WordPress, Squarespace, Jekyll, Stackoverflow, etc. Y pude ser implementado en Perl, PHP, Python, Ruby, Java y Common Lisp.

Dicho lenguaje puede ser convertido a documentos XHTML, mediante la erramienta html2text.
