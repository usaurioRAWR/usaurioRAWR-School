XHTML es una versión de HTML más estricta y basada en XML.

¿Qué es XHTML?

  • XHTML son las siglas de EXtensible HyperText Markup Language
  • XHTML es una versión de HTML más estricta y basada en XML
  • XHTML es HTML definido como una aplicación XML
  • XHTML es compatible con todos los navegadores principales

"http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">

	<html xmlns="http://www.w3.org/1999/xhtml">
		<head>
			<title> Title of document </title>
		</head>
		<body>
			
			some content here...
			
		</body>
	</html>

Las diferencias más importantes con HTML

<! DOCTYPE>

<! DOCTYPE> es obligatorio

xmlns

El atributo xmlns en <html> es obligatorio

<head>, <title> y <body>

Las etiquetas <head>, <title> y <body> son tambien obligatorias.

Correctamente anidado

Los elementos siempre deben estar correctamente anidados

Elementos cerrados

Los elementos siempre deben estar cerrados

Minúsculas

Los elementos y nombres de atributos siempre deben estar en minúsculas

Entrecomillas

Los valores de atributo siempre deben estar entrecomillados

 Minimización prohibida

La minimización de atributos está prohibida
